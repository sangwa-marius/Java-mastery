| Topic                  | Key Point                                  |
| ---------------------- | ------------------------------------------ |
| Java                   | Object-oriented, high-level language       |
| JVM                    | Executes bytecode                          |
| JRE                    | Runtime environment                        |
| JDK                    | Development kit (includes JRE and tools)   |
| `javac`                | Compiles `.java` files into `.class` files |
| `java`                 | Runs compiled classes                      |
| `System.out.println()` | Prints text followed by a newline          |
| `main()`               | Program entry point                        |
| `//`                   | Single-line comment                        |
| `/* */`                | Multi-line comment                         |
| `/** */`               | Documentation comment                      |
